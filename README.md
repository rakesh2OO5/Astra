# Astra 

Astra is a production-ready AI chat application featuring persistent conversation threads, Markdown rendering, syntax-highlighted code blocks, typing animations, and cloud deployment using the MERN stack and OpenRouter API.

## рҹҡҖ Installation

### Clone Repository

```bash
git clone https://github.com/rakesh2OO5/astra.git

cd astra
```

---

### Backend Setup

```bash
cd backend

npm install

npm start
```

Backend runs on:

```text
http://localhost:6767
```

---

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
