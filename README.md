## рҹҡҖ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/astra.git

cd astra
```

---

### Backend Setup

```bash
cd backend

npm install

npm start
```

Backend runs on:

```text
http://localhost:6767
```

---

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
